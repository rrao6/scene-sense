SceneIQ
System architecture

Verified scene prompts, generated from grounded sources

A faithful view of how SceneIQ turns Tubi Moments VLM output into validated prompts, deterministic eval results, client output, and downstream learning signals.

1. Input

Tubi Moments VLM JSON

Scenes, cast, dialogue, themes, objects, audio-recognized songs, and wiki-matched trivia.

→

2. Ingest + domain detection

Prepare eligible scenes

Classify legal, historical, medical, military, hacking, rom-com, etc. Skip flagged or very short scenes.

→

3. Generate in parallel

Five specialized pipelines

Each pipeline uses a defined source/capability and emits prompt candidates with evidence.

Source legend

VLM-only: no model/search call
Gemini: drafting/reasoning
Search grounding: Google-cited URLs
Fetch verifier: checks page text
Official/primary: law, academic, original text
Trusted press: major/trade editorial

Tier-0

Extracts wiki trivia, song IDs, and detected cast already present in VLM output.

Uses: VLM-only deterministic extraction

Cast enrichment

Collects unique actors, uses Gemini with search grounding, keeps only cited evidence.

Uses: Gemini + Google Search grounding

Trivia

Proposes scene topics, searches/fetches sources, generates MCQs, retries bad distractors.

Uses: Gemini + Search + fetched pages

How Real Is It?

Finds what is wrong, right, or invented using official/primary domain sources.

Uses: official/primary sources + fallback search

Facts

Builds editorial BTS prompts and follow-ups from trusted press and title-level topics.

Uses: Gemini + trusted press sources

4. Validators

Hard gates at generation time

Grounding URLs only, verbatim evidence match, cross-source corroboration, source blacklist, expert gate, MCQ option checks.

5. Finalizer

Dedup + scorecard

Dedup across pipelines and score accuracy, legal safety, prompt quality, UX, and monetization fit.

6. Eval

Live re-fetch

Re-fetch cited URLs, verify title grounding, and re-match quotes before final output.

7. Client output

UI bundle + audit

ui.json contract, final.json evidence, review.json scorecard, and human review sheet.

Feedback loop: prompt opens, actor clicks, source expands, title jumps, dwell, skips, and watchlist adds feed ranking features, user profiles, recommendations, and ads/partner intelligence.
