Source-grounded system architecture

SceneIQ: verified contextual prompts plus a feedback loop for ML and ads

A faithful architecture view: what each generation pipeline uses, how claims are validated, where deterministic eval runs, and how post-click interaction becomes downstream signal.

1. Input

Tubi Moments VLM JSON

Scenes, cast, dialogue, themes, objects, songs, wiki-matched trivia.

→

2. Ingest + domain detection

Prepare eligible scenes

Classify legal, historical, medical, military, hacking, rom-com, etc. Skip flagged or very short scenes.

→

3. Generate in parallel

Five specialized pipelines

Each pipeline has a defined source/capability and returns prompt candidates with evidence.

Source/capability legend

VLM-only: no model/search call
Gemini: drafting/reasoning
Search grounding: Google-cited URLs
Fetch: verify fetched page text
Primary/official: law, academic, original texts
Trusted press: trade/major editorial sources

Tier-0

Extract wiki trivia, recognized songs, and detected cast already present in the VLM payload.

Source: VLM-only deterministic extraction

Cast enrichment

Collect unique actors, ask Gemini for grounded actor facts, keep only cited evidence.

Source: Gemini + Google Search grounding

Trivia

Propose scene topics, search/fetch sources, generate MCQs, retry on distractor overlap.

Source: Gemini + Search + fetched pages

How Real Is It?

Find what the scene gets wrong/right/invented using primary or official domain sources.

Source: curated primary/official library + fallback search

Facts

Cluster title-level BTS topics and build editorial prompts with follow-up chains.

Source: Gemini + trusted press sources

4. Validators

Hard gates at generation time

Grounding URL provenance, verbatim anchor match, cross-source corroboration, expert gate, source blacklist, MCQ option-kind checks.

5. Finalizer

Dedup + 7-dim scorecard

Dedup by scene/primitive/keyword hash. Score accuracy, legal safety, prompt quality, UX, and monetization fit.

6. Deterministic eval

Live re-fetch + quote match

Re-fetch cited URLs, verify title grounding, re-match evidence, override generator self-claims.

7. Client output

UI bundle + audit files

ui.json contract, final.json audit, review.json scorecard, and human review prompt sheet.

Post-click feedback loop: actor clicks, prompt opens, source expands, title jumps, dwell, skips, and watchlist adds feed ranking features, richer user profiles, recommendations, and ads/partner intelligence.
