# Competitive landscape — open research

**Owner:** Rahul Rao.

## Hypothesis

Tubi as a FAST challenger has more UX flexibility than SVOD incumbents to redefine the in-player paradigm. The monetization incentive is stronger for AVOD/FAST (interactive depth → new inventory) than for SVOD (conversational search is a retention play, not a revenue play).

## Incumbents to track

- **Amazon Prime Video X-Ray** — cast/music/trivia overlay. Mature product. Pause-screen integration. Baseline to beat on primitives but no conversational layer and limited monetization.
- **Netflix** — occasional interactive (Bandersnatch) but no persistent contextual layer. Investing in AI search/discovery.
- **Hulu / Disney+** — limited in-player context beyond title-level metadata.
- **Roku Channel / Pluto / FAST peers** — mostly no in-player intelligence yet.
- **YouTube** — users leak here today for shoulder content. Reclaiming this is a big lever.
- **Reddit / IMDB / Fandom** — where curiosity currently terminates.

## Differentiation narrative for Tubi

1. **Scene-aware interactive ad formats on CTV** — extends Interactive Pause with contextual premium inventory peers don't have.
2. **Real playback interaction value to the user** — pause stops being dead air (or a generic ad); viewer gets facts, cast, explainers, trivia tied to what they just watched.
3. **AVOD-aligned monetization model** — interactive depth directly unlocks new direct-sold inventory; SVOD peers don't have the same incentive.
4. **Intent-graph data moat (compounding outcome)** — every prompt tap is a structured signal, feeding ranking/personalization over time.
5. **Shoulder content loop (Project Encore)** — P2+ compounding retention play; no peer has this integrated.
